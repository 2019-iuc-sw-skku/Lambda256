# (가제) SKKoin 결제 시스템 - Application codes
## 개발 목표
성균관대학교 내 행사 및 학생 활동 참여에 따른 보상 대체
블록체인 캠퍼스 구현의 기반 구현

## 개발 내용
회원 가입 및 주소 발급, 결제, 송금, 닉네임 기능 구현
